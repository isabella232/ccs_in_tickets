# CCs in Tickets App

## Description:

Notifies the agent when CCs are present in the ticket.

## App location:

* Ticket sidebar
* New Ticket sidebar

## Features:

* Flashes when a CC is added to the ticket
* Flashes when a ticket is opened and there are existing CCs
